## Project Name

**DigiTools – Digital Tools Platform**

---

## Description

DigiTools is a modern web-based platform that provides a collection of powerful digital tools for creators, professionals, and businesses. It offers a clean UI, responsive design, and user-friendly experience to help users work smarter and more efficiently.

---

## Technologies Used

* React.js
* Tailwind CSS
* DaisyUI
* Lucide React Icons

---

## Features

###  1. Responsive Pricing Section

* Clean and modern pricing cards
* Fully responsive across all devices
* Highlighted "Most Popular" plan

###  2. Interactive UI Components

* Dynamic card selection
* Styled buttons with hover effects
* Clean layout using Flexbox & Tailwind

###  3. Modern Footer Section

* Multi-column footer layout
* Social media integration
* Professional design with copyright section
